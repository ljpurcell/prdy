# prdy
A command line tool to check for debug code (`console.log()` for JavaScript, `dd()` for Php, etc.) and run the test suite before submitting a pull-request.

Images and details for `version-1]` are below. The first version was used to explore the problem space and work out any sticking points.

The second version is currently under construction makes use popular libraries to enhance functionality and user-experience.
